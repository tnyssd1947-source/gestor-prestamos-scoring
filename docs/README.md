# Gestor de Préstamos con Scoring Básico

Proyecto académico desarrollado en la materia Ingeniería de Software.

## Descripción

Sistema para la gestión de préstamos personales que incluye:

- Registro de clientes
- Evaluación automática mediante scoring por reglas
- Aprobación o rechazo de solicitudes
- Generación de plan de pagos
- Control de mora y recargos
- Generación de reportes administrativos

## Estructura del Proyecto

- `/docs` → Documentación y entregables
- `/src` → Código fuente del sistema
- `/tests` → Casos de prueba y evidencias
- `/db` → Modelos y scripts de base de datos
